## Получение токенов GigaChat (DEPRECATED)

1) Для начала авторизируйтесь через Сбер ID по данной ссылке https://developers.sber.ru/studio/workspaces

2) Создайте новый API. Укажите имя workspace'a.

3) Перейдите в настройки API. Создайте новый ключ доступа. Скопируйте его и вставьте в .env GIGACHAT_AUTH_KEY.

4) Скачайте файл по <a href="https://gu-st.ru/content/lending/russian_trusted_root_ca_pem.crt">ссылке<a>

5) Создайте папку *cerfs*, вставьте скаченный сертификат

6) Получите ваш токен (валидный в течение 30 минут).
