# T2 — MVP Stakeholder Summary

**AI-Powered Route Optimization Platform | Февраль 2026**

---

## Что такое T2

T2 — платформа оптимизации маршрутов розничной сети, использующая две LLM-модели (Qwen и Llama) для интеллектуальной генерации маршрутов с автоматическим fallback, бенчмаркингом и аналитикой реального времени. Все модели работают локально — данные не покидают инфраструктуру.

---

## Ключевые достижения MVP

| Достижение | Детали |
|------------|--------|
| **15 API-эндпоинтов** | Оптимизация, локации, маршруты, метрики, бенчмарки, insights |
| **2 LLM-модели** | Qwen (основная) + Llama (fallback), автоматическое переключение |
| **4 UI-страницы** | Home, Dashboard, Optimize, Analytics |
| **Docker-инфраструктура** | 4 сервиса запускаются одной командой |
| **CI/CD** | Автоматические тесты, линтинг и coverage через GitHub Actions |

## Бизнес-ценность

- **Снижение стоимости логистики** — оптимизированные маршруты сокращают расходы на топливо и время в пути (ожидаемая экономия 15-20%)
- **AI без облака** — модели работают локально, конфиденциальные данные остаются внутри инфраструктуры
- **Автоматический fallback** — если одна модель недоступна, система переключается на другую без простоя
- **Data-driven решения** — бенчмаркинг и аналитика помогают выбирать оптимальную модель для каждого сценария
