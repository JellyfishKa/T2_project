# Инструкция по записи демо — T2 MVP

## 1. Рекомендуемые инструменты записи

| Инструмент | Платформа | Стоимость | Рекомендация |
|------------|-----------|-----------|--------------|
| **OBS Studio** | Windows / Mac / Linux | Бесплатно | Лучший вариант для качественной записи |
| **Xbox Game Bar** | Windows 10/11 | Встроен | Самый простой — Win+G для начала |
| **QuickTime Player** | macOS | Встроен | Простой, без установки |
| **ShareX** | Windows | Бесплатно | Лёгкий, поддержка GIF |

## 2. Настройки записи

### Разрешение и формат

| Параметр | Значение |
|----------|----------|
| Разрешение | 1920x1080 (Full HD) |
| Формат | MP4 (H.264) |
| Частота кадров | 30 fps |
| Битрейт видео | 4000-6000 kbps |
| Аудио | AAC, 128 kbps |

### Подготовка экрана

- [ ] Установить масштаб браузера на 100% (Ctrl+0)
- [ ] Закрыть все уведомления (Focus Assist / Do Not Disturb)
- [ ] Закрыть лишние вкладки и программы
- [ ] Увеличить шрифт терминала до 14-16pt для читаемости
- [ ] Подготовить все URL в закладках браузера:
  - `http://localhost` — Фронтенд
  - `http://localhost:8000/docs` — Swagger UI
- [ ] Развернуть терминал с curl-командами (см. [MVP_DEMO_SCRIPT.md](MVP_DEMO_SCRIPT.md))

### Аудио

- [ ] Проверить микрофон перед записью
- [ ] Использовать внешний микрофон или гарнитуру (не встроенный в ноутбук)
- [ ] Записать тестовые 10 секунд и проверить качество звука
- [ ] Устранить фоновый шум (закрыть окна, выключить вентилятор)

## 3. Сценарий записи

Повторяет [MVP_DEMO_SCRIPT.md](MVP_DEMO_SCRIPT.md). Ключевые точки:

| Время | Действие | Что на экране |
|-------|----------|---------------|
| 0:00 | Вступление | Титульный слайд или Home-страница |
| 0:30 | Health check | Терминал с curl-командой |
| 2:00 | Создание локаций | Swagger UI или терминал |
| 4:00 | Оптимизация маршрута | Swagger UI → ответ с маршрутом |
| 7:00 | Dashboard UI | Браузер — фронтенд |
| 10:00 | Бенчмарк моделей | Терминал → результаты сравнения |
| 13:00 | Analytics страница | Браузер — графики и статистика |
| 15:00 | Завершение | Резюме ключевых тезисов |

### Советы по записи

1. **Говорите медленно и чётко** — запись можно ускорить, но нельзя замедлить без потери качества
2. **Делайте паузы** между шагами — 2-3 секунды для монтажа
3. **Двигайте курсор плавно** — избегайте резких перемещений
4. **Озвучивайте действия** — «Сейчас я перехожу на страницу Dashboard...»
5. **Если ошиблись** — сделайте паузу и повторите действие; ошибку можно вырезать при монтаже

## 4. После записи

### Проверка

- [ ] Просмотреть запись целиком
- [ ] Проверить качество звука
- [ ] Убедиться что текст в терминале читаем
- [ ] Проверить что нет утечки конфиденциальных данных (пароли, токены, IP)

### Монтаж (при необходимости)

Бесплатные видеоредакторы:
- **DaVinci Resolve** — профессиональный, бесплатный
- **Clipchamp** — встроен в Windows 11
- **iMovie** — встроен в macOS

### Куда сохранить

| Хранилище | Когда использовать |
|-----------|-------------------|
| Google Drive / Яндекс.Диск | Для команды — расшарить ссылку |
| GitHub Releases | Для привязки к версии MVP |
| YouTube (unlisted) | Для удобного просмотра stakeholders |
| Локально | Как бэкап |

### Как поделиться

1. Загрузить видео на выбранное хранилище
2. Получить ссылку для просмотра
3. Отправить ссылку через Slack/Email (использовать шаблон из [MVP_ANNOUNCEMENT_TEMPLATE.md](MVP_ANNOUNCEMENT_TEMPLATE.md))
4. Добавить ссылку на видео в `README.md` проекта (опционально)

## 5. Быстрый старт — запись через Xbox Game Bar (Windows)

```
1. Нажмите Win+G → откроется Game Bar
2. Нажмите кнопку записи (или Win+Alt+R)
3. Проведите демо по сценарию
4. Нажмите Win+Alt+R для остановки записи
5. Видео сохраняется в: C:\Users\[Имя]\Videos\Captures\
```
